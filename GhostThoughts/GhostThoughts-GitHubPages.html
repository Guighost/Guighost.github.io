GitHub Pages - as a free hosting solution

By GuiGhost  posted April 22, 2017 

                 Do yourself a favor and read up on GitHub Pages where you can basically store static web content YOU control for free as long as you are open source. Setup was easy and since you can use javascript, my learning project indy games on the GuiGhost Games page are hosted there. 

            Aspiring developers looking for easy hosting - checkout GitHub Pages. If you own your own domain, you can also use that while hosting the content there. Like any other Git respository, you can use your IDE of choice to publish directory to your site. Below are a couple of resources for getting up and going with your own custom HTML, CSS, and Javascript pages that you have complete control over.
