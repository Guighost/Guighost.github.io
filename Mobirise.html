Trying out Mobirise website builder

By GuiGhost posted April 17, 2017

"Make your own website in minutes", that was advert. Mobirise is a drag and drop, edit, and publish application that you download and install. Well I decided to try it out and this page is a product of the app.

The App: Mobrise comes with some built in templates, a few more you can download, and then the more complex "blocks" can be purchased individually or as a package. This relies mostly on Bootstrap templates and an engine that installs as a a local application. The setup was easy and quick. After just a few minutes of install and setup,I was able to quickly create and publish static content to github pages with ease, using a drag and drop interface. Users are able to manage multiple site projects, and publish each one separately. Other options include the ability to save custom "blocks" that can be used as templates. You could, for instance use the application as an easy publisher  for a blog post or informational article, without the need to know coding.

  User Community:  Next I delved into the user community available. I found several sites that have additional "User Blocks" to be imported using the Extensions menu. Once done, these items are added into the drag and drop site builder menus, expanding the free content. I have tried out several already. Here are a few of the sites that host addional Mobirise user blocks for download, both paid and free.

  • Mobirise Themes
  • Unicore Theme
  • Mobirise Icons
  • Mobirise Market
  • Forum

         As I delve into this further, I will update this article with more findings on this tool that can be used as a quick start to simple websites. You might want to check it out if you are interested in a no coding approach to building a mobile friendly simple website for small to medium businesses. Stay tuned for more or check it our yourself at mobirise.com!
